O curso

Conteúdo

O curso consiste em implementar uma aplicação Java-Web passo a passo utilizando recursos inovadores da plataforma Java desde a versão 8 até a 15. Alguns tópicos: modularização, records, lambda, objetos anônimos, etc.

Java

No curso usaremos a versão 15 do Java. Utilizando o que há de mais atual na plataforma Java como modularização, jlink, records e etc. Entretanto o requisito básico é ter noções de Java, ou seja, não é necessário nenhum conhecimento específico de Java ou de seus frameworks.

Spring

No curso faremos uso intensivo do principal e mais utilizado framework Java o Spring. Novamente o requisito básico é ter noções de Java, ou seja, não é necessário nenhum conhecimento específico de Java ou de seus frameworks.
